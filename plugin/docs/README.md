## Document Purposes

### WORKFLOW.md
The complete CipherPowers workflow guide. Covers:
- When to brainstorm, plan, or execute directly
- Batch execution with code review checkpoints
- Verification commands and dual-agent review
- Full examples from start to finish

### COMMANDS.md
Reference for all `/cipherpowers:*` slash commands:
- `/cipherpowers:brainstorm` - Design refinement
- `/cipherpowers:plan` - Implementation planning
- `/cipherpowers:execute` - Plan execution
- `/cipherpowers:code-review` - Code review
- `/cipherpowers:commit` - Atomic commits
- `/cipherpowers:verify` - Dual verification
- `/cipherpowers:summarise` - Learning capture

### SKILLS.md
Catalog of all skills organized by category:
- Development skills (TDD, debugging)
- Planning & execution skills
- Code review skills
- Documentation skills
- Workflow skills
- Meta skills

### AGENTS.md
Reference for specialized subagents:
- Development agents (code-agent, rust-agent)
- Review agents (code-review-agent, plan-review-agent)
- Support agents (commit-agent, technical-writer)
